Preparation for Your Blog


Reminder: you can skip the post if you merely want to know how to build a blog on GitHub.

Developing a personal website has been on my To-Do list for a long time. In my opinion, there are four tasks most web designers will care about:

  1. Reliable Web Hosting
  2. Memorable domain name
  3. Management Platform
  4. Artistic Web Design


As far as I am concerned, good visual design is simple and clean style. Artistic design can keep refining all the time, however, the first three tasks need to be decided at very beginning stage. That is why I want to share my experience of choosing web host, domain name service, and management platform with readers. Welcome any idea and suggestion from you. YOUR ADVICE IS PRICELESS!

Web Hosting

Before I know it is possible to build a static blog on GitHub, I have already spent some time on free web hosts research. The free web hosting directory is one useful review website for those still looking for good free web hosts.

After reading some review, I noticed experienced website designers cared about the down time of service most. Sometimes FREE is more expensive because of frustrating unexpected down time. Thus, paying for a web hosting in an affordable price might not be a bad choice for the easy installation, website stability and technicial support. HostGator, bluehost and justhost are frequently mentioned with positive reviews on internet.

If you are a IT professional, renting a virtual private server (VPS) is another decent choice. cloudatcost recently offers several plans with attractive price for light and heavy users. You can give it a try his basic plan:

$1/month or $35/lifetime for 1CPU, 512MB, 10GB storage, 500GB monthly transfer

Domain name

Top-level domain name such as .com, .net, and .org usually costs $10 to $20 per year. Of course! Some companies provide you FREE second-level domains. For example, GitHub gives [your user id].github.io for your blog. If you do not like .github.io, here is a list of well-known domain registrars:

  • Godday is the most popular one and often does promotion such as $.99/first year
  • Namecheap has better user-friendly interface compared to Godaddy

Management Platform

Some people mention that Wordpress is the best blog platform. It not only offer 3GB free online space but also has user-friendly tools to build a personalized blog easily. Besides Wordpress, Joomla and Drupal are top 2 and 3 content management systems (CMS) boardly used for community and businesss websites in the world. To be honest, Drupal + VPS had been my original choice for my website until I was informed that GitHub has GitHub pages feature for developers to build a simple static blog.

The main reason that I choose GitHub as my first step of learning web design is:

  1. No worry about down time.
  2. Free GitHub Pages service
  3. I can learn Git, Markdown, and Node.js during creating my blog.

You cannot wait to learn how to use GitHub any more, can’t you? Let’s move on~
